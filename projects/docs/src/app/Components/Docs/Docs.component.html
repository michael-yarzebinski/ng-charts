Axis Classes

Title
Property Type Description
Show boolean Determines whether the Axis Title is shown or not.
Style Text Determines the styling of the Axis Title.
Space number How wide/tall (depending on orientation) the Axis Title box is. This allows for custom spacing.
Text string The Axis Title.
Tick Marks
Property Type Description
Show boolean Determines whether the Tick Marks are shown or not.
Style Line Determines the styling of the Tick Marks.
Size number How wide/tall (depending on orientation) the Tick Marks are.
Labels
Property Type Description
Show boolean Determines whether the Labels are shown or not.
Style Text Determines the styling of the Labels.
Space number How wide/tall (depending on orientation) the Label boxes are. This allows for custom spacing.
Format Function Each label will be passed through this function. This allows for custom label formats.
Grid Lines
Property Type Description
Show boolean Determines whether the Grid Lines are shown or not.
Style Line Determines the styling of the Grid Lines.
Axis
Property Type Description
Show boolean Determines whether the Axis is shown or not.
Position Left, Right, Top, or Bottom Determines the position of the Axis. It is up to the users to use them properly (X Axis is Top or Bottom, Y Axis is Left or Right).
Title Title Information about the Title.
Grid Lines Grid Lines Information about the Grid Lines.
Tick Marks Tick Marks Information about the Tick Marks.
Labels Labels Information about the Labels.
Min any The minimum value associated with the Axis.
Max any The maximum value associated with the Axis.
Tick Count number Number of Ticks on the Axis. d3 does it's best to accomidate to this.
Reverse boolean Whether the axis is ascending or descending.

Series Classes

Point
Property Type Description
X any X value of a Point on a graph.
Y any Y value of a Point on a graph.
Scatter Point (extends Point)
Property Type Description
Radius any Radius of a Scatter Point on a graph.
Tooltip Properties
Property Type Description
Template TemplateRef<ElementRef> Template to be used for Tooltips. Note: You may extend the Point/Scatter Point classes to provide additional information in the Tooltip.
Show boolean Whether to display the Tooltip or not.
Style Polygon Styling of the Tooltip.
CustomClass string Apply a Custom Class to the Tooltip.
Position Top, Left, Right, or Bottom Where the Tooltip should be placed. If one is not provided, the best position will be determined (default is Top).
Series
Property Type Description
Name string Name of the series.
Data Array of Points of Scatter Points Data associated with a given series.
Y Axis Left or Right Which Y Axis the Series corresponds to.
Type Area, Line, or Scatter Read Only property. Set in constructor.
Style Line or Polygon Information about the coloring of the SVG Line or SVG Path.
Show boolean Determines if the Series should be displayed.
Tooltip Tooltip Properties Information about the Tooltip for the given Series.
Legend Transform string This is used to position the Series in the Legend. Any string provided will be overwritten.

Area, Line, and Scatter Series all implement the Series interface and set appropriate properties in constructor.

Helpful Functions

Build Defaults

Build Default Axis
Property Type Description
Title string Title of the Axis.
Color string Applies the Color to the Color and Fill properties of the Title, Labels, and Tick Marks. Still uses light grey for grid lines.
Position Left, Right, Top, or Bottom Applies to the Position property of the Axis.
POrS (Primary or Secondary) Primary of Secondary Primary shows the Grid Lines, Secondary shows the Tick Marks.
Build Default Series
Property Type Description
Name string Name of the Series.
Data Array of any Sets the Data property of the Series.
Y Axis Left or Right Sets the Y Axis property of the Series.
Type Area, Line, or Scatter Determines what type of series to build.
Color string Applies the Color to the Color and Fill properties of the Style and Tooltip.
Build Default Legend
Property Type Description
Position Top, Left, Right, or Bottom Sets the Position property of Legend Options.
Orientation Horizontal or Vertical Sets the Orientation property of Legend Options. Also sets a default for Space property.
